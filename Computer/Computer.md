## Traversal

### Tree Traversal

Tree of Alphabets
```shell
    B
A       C
```

#### Pre-Order Traversal

Root - Left - Right
```shell
B - A - C
```

#### In-Order Traversal

Left - Root - Right
```shell
A - B - C
```

#### Post-Order Traversal

Left - Right - Root
```shell
A - C - B
```
