Pour utiliser cette version Winhelp de la documentation
du JSDK 2.3, dé-zippez le fichier dans un répertoire
quelconque de votre disque dur.

Il ne reste plus alors qu'à double-cliquer Jsdk23.hlp
pour lancer l'aide.

Mettre un raccourci sur Jsdk23.hlpdans votre menu
Démarrer ou sur le bureau est un bonne idée, je crois...

Utilisez l'icône présente dans la distribution comme vous le voulez.
En changeant l'icône du raccourci, par exemple...

A propos de HtmlToHlp
---------------------

Les fichiers HTML, téléchargés depuis le site JavaSoft, ont été convertis
au format RTF de Microsoft avec HtmlToHlp, une application Java de
conversion HTML vers WinHelp. 

Vous pouvez utiliser HtmlToHlp pour convertir vos propres documentations HTML
en fichiers WinHelp : ce logiciel est freeware pour une utilisation personnelle,
et shareware 150,00 Francs Français (US$ 25.00) pour une utilisation commerciale.

Pour toute information complémentaire, rendez-vous
sur http://www.confluent.fr/javadoc.
